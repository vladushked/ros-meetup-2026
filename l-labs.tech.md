# l-labs.tech

## Информация от меня

Компания занимается разработкой промышленного робота-сортировщика ТКО (мусора и отходов). 
Цель — полная автоматическая сортировки отходов. 
Описание робота - промышленный робот-манипулятор портального типа с 4-мя степенями свободы (перемещение по XYZ и вращающийся вокруг вертикальной оси хват). Робот использует rgbd систему тех зрения и детектор объектов для распознавания мусора. ПО робота написано на базе ROS2.
Принцип работы - робот и модуль технического зрения (МТЗ) устанавливаются над конвейерной лентой. При первоначальной настройке выполняется калибровка МТЗ и проверка всех подсистем робота. Робот управляется через панель оператора, на которой выбираются фракции для сортировки. Робот может сортировать до 4-х фракций одновременно. Из МТЗ поступают изображения в ПО верхнего уровня, далее стек технического зрения выполняет детекцию и трекинг объектов, определяет точки и углы захвата для каждого объекта, затем алгоритм управления выбирает наиболее релевантный объект и выполняет захват и сортировку объекта в нужную урну.
