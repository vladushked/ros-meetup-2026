# ROS Meetup 2026 Materials

Этот каталог содержит:
- `presentation.md` — актуальный план презентации;
- `tutorials/` — пошаговые туториалы по каждой теме презентации;
- `examples/` — полноценные пример-файлы, на которые ссылаются туториалы.

## Список тем

1. Launch-архитектура и прокидка аргументов
2. Система сборки и запуска (build/run/completion/docker)
3. Базовый класс ноды (sync/async)
4. Runtime изменение и сохранение параметров
5. Разделение конфигов для нескольких роботов
6. Asyncio в ROS 2
7. `register_loop` и несколько loop в одной ноде
8. Асинхронная FSM
